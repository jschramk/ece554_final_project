Instructions

This tool is for creating Verilog modules from the specification detailed in the Coding! at the Disco Microarchitecture Review Presentation. It will not work with anything else.

1) Copy the name of a module from one of the Google Slides and paste it into the "Module Name" input. (do not worry about copying extra spaces)

2) Copy the first two columns of the table describing the module ports, making sure to copy all rows containing ports. DO NOT copy the first row containing the column titles.

3) Paste the copied data directly into the top text area as it is.

4) Click "Convert" and make sure the output in the bottom text area looks correct.

5) Repeat steps 1 - 4 for as many modules as needed.

6) Click "Download All" to download all the generated Verilog files.
